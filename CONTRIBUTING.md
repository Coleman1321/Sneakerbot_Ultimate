# Contributing to SneakerBot Ultimate

Thank you for your interest in contributing to this security research project!

## Code of Conduct

This is an educational/security research project. Contributions must:
- Focus on educational value
- Improve security research capabilities
- Document attack vectors and defenses
- Follow ethical guidelines

## How to Contribute

1. **Fork the repository**
2. **Create a feature branch** (`git checkout -b feature/improvement`)
3. **Make your changes** with clear documentation
4. **Test thoroughly**
5. **Commit** with descriptive messages
6. **Push** to your fork
7. **Submit a Pull Request**

## What We're Looking For

✅ **Documentation improvements**
✅ **Bug fixes**
✅ **New attack vector demonstrations**
✅ **Enhanced defensive recommendations**
✅ **Code quality improvements**
✅ **Test coverage**

❌ **Commercial bot features**
❌ **ToS circumvention tools**
❌ **Unethical use cases**

## Code Standards

- Follow PEP 8 for Python code
- Include docstrings for all functions
- Add comments for complex logic
- Update documentation as needed
- Write tests when applicable

## Security Research Guidelines

- Document all attack vectors clearly
- Provide defensive countermeasures
- Emphasize educational purpose
- Include ethical considerations
- Test responsibly

## Questions?

Open an issue for discussion before making major changes.
